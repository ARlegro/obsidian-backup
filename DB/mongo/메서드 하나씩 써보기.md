### 필드 수정 연산자 

#### $inc ➡ 내부의 값 1씩 증가시키는 로직 

```mongodb
{ $inc: { 필드 : 숫자 }}
```
- 필드의 값을 증가시키거나 감소시킬 수 있다
- ex.
	- 숫자=1 ➡ 필드값 1증가
	- 숫자=2 ➡ 필드값 2증가
	- 숫자= -1 ➡ 필드값 1 감소 

>[!tip] 특정 필드값을 가진 doc을 상대로 연산 
>- update_one(filter, update)  
>	- filter : 업데이트 할 문서를 찾는 조건 
>	- update : MongoDB 연산을 통한 코드 
>	- 반환 값 : 
```python
    # 2) likes 필드를 1 증가
    result = db.movie.update_one(
        {"title": title},                          # 찾을 문서 조건
        {"$inc": {"likes": 1}}                     # likes += 1
    )
```
