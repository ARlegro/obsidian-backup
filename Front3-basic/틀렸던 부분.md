>[!EXAMPLE] 포스팅 박스 만들기

![[Pasted image 20250512140435.png]]
```css
#posting-box {
  border: 3px solid black;
  border-radius: 5px;
  
  width: 400px;
  margin: 10px auto 20px auto;
  
  padding: 50px;
}
```
- border를 3px 이렇게 적용안하니 안보였다.
- padding : 내용이 테두리에 붙지 않게 하기 위해!!
