##### 74. 다음 C언어 프로그램이 실행되었을 때의 결과는?

```C
int main(int argc, char *argv[]) {

```

>[!EXAMPLE] C언어의 Bitwise OR 연산 ( | )
>- 2비트로 바꾼 뒤 연산
>- OR 연산자는 두 비트 중 1개라도 1이면 1이 출력된다
>```C
int a = 4;      // 2진수로: 0000 0100
int b = 7;      // 2진수로: 0000 0111
int c = a | b;  // 비트 OR 연산  
  0000 0100   (a = 4)
| 0000 0111   (b = 7)
  0000 0111   → 결과: 7
