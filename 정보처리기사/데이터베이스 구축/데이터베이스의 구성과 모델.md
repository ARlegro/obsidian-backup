
> 스키마란?
> - 데이터베이스 구조(개체, 속성, 관계 등)에 대한 정의

### 스키마의 3계층 ⭐⭐

1. 외부 스키마
2. **개념 스키마** ⭐
	- **DB 전체를 정의**한 것  
	- 가장 중요한 스키마
	- 얼마나 효율적으로 잘 구성해서 효과적으로 관리할지를 정하는 스키마
	  
3. **내부 스키마** 
	- 데이터의 실제 저장 방법을 기술한 것 
	- 물리적 저장장치의 입장에서 본 DB 구조 

### DB 3단계 언어

1. 데이터 정의어(DDL)
2. 데이터 조작어(DML)
3. 데이터 제어어(DCL)
	- 데이터 권한 보호
	- 무결성 유지
	- 데이터 회복 및 병행 제어 수행
