### 개요 
여기서의 코드란, 데이터의 사용 목적에 따라서 식별, 분류, 배열하기 위하여 사용하는 숫자-문제-기호를 의미 

**주요 목적** 
- 고유성 : 코드와 뜻이 1:1로 확실하게 대응되어야 한다
- 분류 편의성 : 목적에 적합한 분류가 가능해야 한다


### 기능 
1. **코드의 기본적 기능** 
	- 표준화 기능
	- 간소화 기능 
	- 즉, 쉽게 접근 가능해야 한다 
	  
2. **코드의 3대 기능**
	- 분류 기능 : 자료를 분류
	- 식별 기능
	- 배열 기능
3. **코드의 부가적 기능** 
	- 연상 기능 ex. 변수명 보고 추측 
	- 암호화 기능 
	- 오류 검출 기능
